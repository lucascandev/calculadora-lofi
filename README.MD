## Calculadora Lo-Fi v0.01 ~~ Lucas

O intuito do projeto é destinado criar uma calculadora dinâmica, onde o usuário pode utilizar as funções e visualizar seu histórico de cálculos, tudo isso em linguagem Python.

## Estrutura do que busco implementar nesse projeto:

- Interação do Usuário:
O usuário pode realizar cálculos básicos, tanto como soma, subtração, divisão e multiplicação e etc...

- Interface:
A idéia é criar uma interface gráfica usando Tkinter para tornar o projeto mais dinâmico.
